######################
使用说明：
############################
1:需要将该插件引入到对应的项目中
############################
2:在项目中需要挂载 TabBar 与 TabBarItem 两个组件
####################################
3:在 TabBarItem 中具有两个具名插槽，icon 表示图标字体，text 表示文字内容
#############################3
4:在 App.vue中使用
    <TabBar>
      <TabBarItem>
        <div slot="icon"></div>
        <div slot="text"></div>
      </TabBarItem>
    </TabBar>
###################
5:在TabBarItem 组件上可以设置属性
1.path属性：设置路由路径
2.clo属性：设置点击颜色状态
